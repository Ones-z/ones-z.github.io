三元表达式

在Python中，三元表达式的形式为 value_if_true if condition else value_if_false。这种表达式的作用是根据条件的真假选择两个值中的一个作为整个表达式的结果。
使用三元表达式可以简化某些条件判断的语法，并且可以在一行中完成相应的逻辑操作。这在某些情况下可以提高代码的可读性和简洁性。
